# qr2text v1

## Description

 A program to render a QR code as plain text, using block symbols.

 Based on:

 * [qr2text](https://github.com/wapm-packages/qr2text)


![](qr2text.jpg)

## Dependencies

* [egui](https://github.com/emilk/egui)


## Build and run

```
cargo build --release

cargo run --release
```
